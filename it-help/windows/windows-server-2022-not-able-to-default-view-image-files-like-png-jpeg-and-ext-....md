# Windows server 2022 not able to default view image files like png, jpeg and ext ...

If your Windows Server 2022 isn't displaying PNG images correctly, it's likely due to a missing or corrupted image viewer, incorrect file associations, or potential issues with the PNG file itself; check your default image viewer settings, ensure the PNG file is not corrupted, and if necessary, install a compatible image viewer application to properly view PNG images on your server.
