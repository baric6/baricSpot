# Remove User and pass from Credential Manager



1. **Open Control Panel**: Press the Windows key, type `Control Panel`, or type in the search **Credential Manager**
2. **Go to Credential Manager**: Click on **User Accounts**, then select **Credential Manager**.
3. **View Windows Credentials**: In the Credential Manager window, click on **Windows Credentials** to see a list of saved accounts and passwords.
4. **Remove the credential**: Find the specific credential you want to delete, click on it to expand the details, and then click **Remove**.
5. **Confirm deletion**: A confirmation dialog will appear. Click **Yes** to permanently remove the saved credentials.&#x20;
