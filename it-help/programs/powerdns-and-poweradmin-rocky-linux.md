# PowerDNS and PowerAdmin rocky linux

PowerDNS is a free and open-source DNS Server software. It's can be used as an authoritative nameserver and DNS recursor. The PowerDNS is written in C++ and supports multiple operating systems, including BSD, Linux, and macOS.
