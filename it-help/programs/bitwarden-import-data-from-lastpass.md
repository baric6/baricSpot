# Bitwarden Import Data from LastPass

{% embed url="https://bitwarden.com/help/import-from-lastpass/" %}

To import and export data from LastPass to Bitwarden, you can do the following:

1. 1\. **Export from LastPass**
   1. Log in to your LastPass account
   2. Click Advanced Options in the sidebar
   3. Select Export
   4. Enter your LastPass master password
   5. Download the CSV file to your computer
2. 2\. **Import to Bitwarden**
   1. Log in to your Bitwarden account
   2. Select Tools from the top menu
   3. Click Import Data
   4. Select LastPass (CSV) from the list of file formats
   5. Select the exported CSV file
   6. Click Import Data&#x20;

You can also import data directly from LastPass into Bitwarden:

1. Log in to the Bitwarden browser extension or desktop app
2. Select the Settings tab
3. Choose Vault and then Import items
4. Follow the prompts to import directly from LastPass&#x20;
5.

Bitwarden may reject imports that exceed certain data limitations, such as the number of items, folders, collections, or item-folder relationships.&#x20;
