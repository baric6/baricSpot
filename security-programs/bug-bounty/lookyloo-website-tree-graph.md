# Lookyloo website tree graph

<figure><img src="../../.gitbook/assets/image (235).png" alt=""><figcaption></figcaption></figure>

With Lookyloo you can dissect a website while it is in motion. Lookyloo is a web interface that captures a webpage and then displays a tree of the domains, that call each other.
