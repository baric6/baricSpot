# GoGo port scanner

* Free port configuration
* Support active/passive fingerprint recognition
* Key information extraction, such as title, cert, and custom rules for extracting information
* Support nuclei poc
* Harmless scanning, each added poc is manually reviewed
* Controllable Heuristic Scanning
* Super performance, fastest speed, as small memory and CPU usage as possible.
* The principle of minimum contract delivery, send as few packages as possible to obtain the most information
* Support DSL, you can customize your own gogo through simple configuration
* Perfect output and output design
* Little dependence on third-party libraries, written in pure native go
