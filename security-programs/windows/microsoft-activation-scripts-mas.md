# Microsoft Activation Scripts (MAS)

<figure><img src="../../.gitbook/assets/MAS_AIO-45ebdebc8c72356fe5e4902e8b893dbb.png" alt=""><figcaption></figcaption></figure>

A Windows and Office activator using HWID / Ohook / KMS38 / Online KMS activation methods, with a focus on open-source code and fewer antivirus detection.

#### Method 2 - Traditional

* Download the file from [GitHub](https://github.com/massgravel/Microsoft-Activation-Scripts/archive/refs/heads/master.zip) or [Bitbucket](https://bitbucket.org/WindowsAddict/microsoft-activation-scripts/get/master.zip) <- download links
* Right-click on the downloaded zip file and extract
* In the extracted folder, find the folder named `All-In-One-Version`
* Run the file named `MAS_AIO.cmd`
* You will see the activation options. Follow the on-screen instructions.
* That's all.
