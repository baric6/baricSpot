# PersistenceSniper

<figure><img src="../../.gitbook/assets/image (295).png" alt=""><figcaption></figcaption></figure>

PersistenceSniper is a Powershell module that can be used by Blue Teams, Incident Responders and System Administrators to hunt persistences implanted in Windows machines. It is also available on Powershell Gallery and it is digitally signed with a valid code signing certificate. The tool is under active development with new releases coming out by the week, so make sure to use the up-to-date version.&#x20;
