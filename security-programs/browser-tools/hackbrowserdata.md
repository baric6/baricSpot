# HackBrowserData

<figure><img src="../../.gitbook/assets/LOGO.png" alt=""><figcaption></figcaption></figure>

HackBrowserData is a command-line tool for decrypting and exporting browser data ( passwords, history, cookies, bookmarks, credit cards, download records, localStorage and extension ) from the browser. It supports the most popular browsers on the market and runs on Windows, macOS and Linux.&#x20;
