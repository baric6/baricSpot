# Introducing the REx: Rule Explorer Project

<figure><img src="../../.gitbook/assets/detr-overview.gif" alt=""><figcaption></figcaption></figure>

The **REx** project is a collection and breakdown of several of the most popular open security detection rules for analysis and exploration, enabled by the powerful search and visualization capabilities of the Elastic stack! The docs can be found at [rulexplorer.io](https://rulexplorer.io) .
