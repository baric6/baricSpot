# Rapid diagnosis testing server health

This menu you can run tests on server for networking, cluster, cpu and others to make sure everything is running smoothly

#### How

1. Go to settings > rapiddiag
2. you will see health checks
   1.

       <figure><img src="../../../../.gitbook/assets/image (448).png" alt=""><figcaption></figcaption></figure>
3. Select one and hit the next button on top
4. which ever one you pick it will take you to the settings of the diagnosis you selected
   1. if you need to set something up it will have the button up top grayed out and a triangle warning on what needs to be set up
5. It will take time to run but you should see a Passed or failed when it is done
