# Splunk Developer program



{% embed url="https://dev.splunk.com/developer-program/" %}

#### How to get a License

1. Make a Splunk account from the link above
2. Under the dev license tab there is a dev license button
3. It will give you a questionnaire about what you are going to use it for&#x20;
4. Then submit they said it should take a few days to get it, i got mine in a few hours&#x20;

#### Apply the license

1. You will get a email from Splunk with the .license file
2. Download it&#x20;
3. Go to your Splunk instance under settings > licensing > add license
4. Upload your file&#x20;
   1. (side note) the file is just a XML and you can copy it in the option under upload
5. It will restart your instance
6. You should see it say what license you have with the allowed GB&#x20;
