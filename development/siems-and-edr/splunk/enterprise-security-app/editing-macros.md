# Editing Macros

Macros are a SPL code snip its that are resealable

**Search > Advanced Search > Search Macros**&#x20;

1. find the macro you are searching for
2. A Lot of default macros are just "search \*" these kind of macros are just for filtering&#x20;
3. Replace with below, this is will filter out names

```
NOT (user IN ("svc_account1","svc_account2"))
```

4. Everything else as default for right now
5. Save
