# Linux - Python starting a venv

#### Install venv lib if not already installed

```
apt install python3.12-venv
```

or

```
apt install python3 python3-venv -y
```

#### Create a venv project

```
python3 -m venv venv
```

#### Activate&#x20;

```
source /venv/bin/activate
```

#### Usage&#x20;

```
pip3 install requests
```
